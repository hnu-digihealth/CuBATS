
Image Registration
==================

Module Documentation
--------------------
This module provides a wrapper for WSI registration using `VALIS <https://github.com/MathOnco/valis>`_. It includes registration of WSIs either with or without a reference slide around VALIS's interal registration functions, as well as a more recent customizable high-resolution registration method.
